## 📁 作品一覧

### 🌤️ 天気アプリ
東京の3日分の天気予報を表示するアプリ。

- **使用技術**: HTML / CSS / JavaScript
- **学んだこと**: fetch・非同期処理・API連携・DOM操作
- **使用API**: [Open-Meteo](https://open-meteo.com/)（無料・キー不要）
