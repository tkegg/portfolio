## 📁 実績・プロジェクト (Works)

### 🏋️ Gym Congestion Notifier (ジム混雑通知システム)
自身の「ジムの混雑を避けたい」という課題を解決するため、完全自動の通知システムを構築しました。
- **使用技術**: Python / Playwright / GitHub Actions / LINE Messaging API
- **工夫した点**: ヘッドレスブラウザによる動的サイトのスクレイピング。GitHub Actionsを用いたサーバーレスでの定期実行。機密情報（APIキー等）をSecretsで管理し、セキュアな運用を実現。

### 📰 AI News Summarizer (開発中)
情報収集のタイムパフォーマンスを最大化するニュース要約アプリケーション。
- **使用技術**: Next.js / Claude API (Anthropic) / Tailwind CSS
- **現在の取り組み**: 最新のReactアーキテクチャ（App Router）の学習と、高度な自然言語処理モデルを用いた要約・翻訳機能のAPI連携を実装中。

### 🌤️ 天気アプリ / 🎋 おみくじアプリ
フロントエンド基礎学習時の制作物。
- **使用技術**: HTML / CSS / JavaScript
- **学んだこと**: fetchによる非同期処理、Open-Meteo API連携、DOM操作の基礎。
